# Second Church COCHUSA — Brand Color Guide

## Primary Palette

| Role | Hex | Usage |
|---|---|---|
| **Navy Dark** | `#010048` | Dark section backgrounds, footer, service-times band, newsletter band, hero |
| **Navy Ink** | `#03002e` | Body text, deep ink color |
| **Crimson** | `#970707` | Primary buttons, hero italic accent ("Worship"), interactive highlights |
| **White** | `#FFFFFF` | Light section backgrounds (About, Ministries, Community, Events, Connect) |

## Secondary / Supporting Palette

| Role | Hex | Usage |
|---|---|---|
| **Gold** | `#B8872A` | Section labels, dividers, border accents |
| **Amber** | `#D4A040` | Gold hover states, footer tagline, sand accents |
| **Sand** | `#E8C97A` | Subtle text highlights |
| **Mid Brown** | `#8C7050` | Muted body text, secondary captions |
| **Parchment** | `#F0E6D0` | Subtle background tints |
| **Mist** | `#EDE3D4` | Borders, dividers on light surfaces |

## Hero Gradient

```css
background: linear-gradient(
  135deg,
  rgba(1, 0, 72, 0.85) 0%,
  rgba(1, 0, 87, 0.50) 50%,
  rgba(2, 0, 108, 0.75) 100%
);
```

## Scrolled Navigation

```css
background: rgba(1, 0, 72, 1);
```

## Notes

- The navy dark (`#010048`) pairs directly with the royal blue used in the COCHUSA logo.
- Crimson (`#970707`) echoes the maroon swoosh in the COCHUSA logo.
- White sections provide high contrast against navy backgrounds for readability.
- Gold (`#B8872A` / `#D4A040`) is retained as a supporting accent — not a primary color.
